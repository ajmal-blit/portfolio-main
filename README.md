# Ajmal Kx — Cinematic Portfolio

## Use
1. Upload `index.html`, `style.css`, and `script.js` to the root of your GitHub repository.
2. Keep your existing `assets` folder in the same location.
3. The expected image files are:
   - `assets/main-img.jpeg`
   - `assets/project-1.png`
   - `assets/project-2.png`
   - `assets/project-3.png`
   - `assets/project-4.png`
4. Commit the files and redeploy GitHub Pages/Vercel.

The old misspelling `assests` has been corrected to `assets`.
